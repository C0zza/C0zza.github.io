Mobile Game Development

Mobile Input Focused Mini-Games

This is a selection of mini games designed to demonstrate the use of mobile input. The game contains a series of tasks where the user must follow the instruction on screen and complete it within the given time frame. As the game progresses this time also decreases. If the user loses too many lives, they must restart.

RPG Prototype

Here the same input methods have been used to create a more comprehensive game. As seen in the video, it includes a combat system with different abilities and inputs, an inventory system and a scene manager.
